## EEG Processing Pipeline (MATLAB)

The MATLAB scripts used to process the raw EEG recordings are organized as follows:

| Directory    | Purpose                                      |
|--------------|----------------------------------------------|
| `/ICA`       | EEG import, filtering, ICA decomposition     |
| `/UniGui`    | GUI-based epoch extraction and preprocessing |
| `/EEGQ`      | Spectral power computation and CSV export    |
| `/MergeData` | Merging multiple EDF or SET recordings       |

### Requirements

* **MATLAB R2018b** or later
* **EEGLAB v2023.1** or later

### Execution Order

1. Run **`ICA/ICA_Script.m`** using all `.EDF` files from the experiment.
2. Execute **`UniGui/Unified_GUI.m`** with all `.SET` files generated by `ICA_Script.m`.
3. Run either:

   * **`EEGQ/eeg_q_Script.m`**
   * **`EEGQ/eeg_q_Unified_Region_Script.m`**

     using the `.SET` files produced by `Unified_GUI.m`.

> If you need to merge `.EDF` files before step 1, use **`MergeData/scriptMergeEDF.m`**.
> If you need to merge `.SET` files, use **`MergeData/MergeSET.m`**.
