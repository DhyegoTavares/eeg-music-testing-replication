## Overview

This directory contains the processed data and analysis scripts used to produce the statistical
results reported in the paper. The study compares frontal EEG power (alpha, beta,
delta, and theta bands) between two groups of participants performing software
testing tasks: one group listened to music (Music) and one did not (NoMusic).

## Dataset Description (`data/EEG_ALL_GROUPS.csv`)

| Column      | Description                                            |
|-------------|--------------------------------------------------------|
| Participant | Participant ID (1–7)                                   |
| BL01–BL04   | Mean frontal EEG power (dB) during task blocks 1–4    |
| IBOE        | Baseline (pre-task) frontal EEG power (dB)            |
| Group       | Experimental condition: `Music` or `NoMusic`          |
| Wave        | EEG frequency band: `Alpha`, `Beta`, `Delta`, `Theta` |

- **n = 7** per group (Participant 008 was excluded as an outlier)
- Values represent mean log-power (dB) at the **frontal electrode region**
- Exported from EEGLAB (MATLAB) and preprocessed via `01_preprocess_data.py`

### Storage Requirements

The processed dataset (`data/EEG_ALL_GROUPS.csv`) is less than 10 KB.
The raw EEGLAB exports in `raw_data/` total approximately 200 KB.
No special storage or hardware requirements are needed.

## Analysis Pipeline

Run all scripts from inside the `statistical_analysis/` directory:

```bash
# Step 1 – (Optional) Rebuild the combined dataset from raw EEGLAB exports
python 01_preprocess_data.py --input_dir raw_data/ --output_dir data/

# Step 2 – Normality tests (required before choosing parametric vs. non-parametric)
python 02_normality_tests.py

# Step 3 – Non-parametric group comparison
python 03_mann_whitney_test.py

# Step 4 – Parametric group comparison
python 04_independent_ttest.py
```

Steps 2–4 read from `data/EEG_ALL_GROUPS.csv` and write CSV summaries to `results/`.
`data/EEG_ALL_GROUPS.csv` is already provided, so Step 1 is only needed if you wish
to regenerate it from the raw EEGLAB exports in `raw_data/`.

## Study Design

### Experimental Conditions
- **Music group**: participants listened to background music while performing software testing tasks
- **NoMusic group**: participants performed the same tasks in silence

### Task Blocks
| Code  | Description                    |
|-------|--------------------------------|
| IBOE  | Baseline (pre-task, eyes open) |
| BL01  | Task block 1                   |
| BL02  | Task block 2                   |
| BL03  | Task block 3                   |
| BL04  | Task block 4                   |

### EEG Frequency Bands
| Band  | Frequency range |
|-------|-----------------|
| Delta | 0.5–4 Hz        |
| Theta | 4–8 Hz          |
| Alpha | 8–13 Hz         |
| Beta  | 13–30 Hz        |

---

## Statistical Approach

1. **Normality** verified with the Shapiro-Wilk test (α = 0.05)
2. **Mann-Whitney U** — non-parametric between-group comparison; effect size *r = |Z| / √N*
3. **Independent t-test** — parametric comparison; Levene's test selects Student vs. Welch; effect size Cohen's *d*
4. **Baseline (IBOE)** tested in both analyses to confirm group equivalence prior to tasks

---

## Key Findings Summary

| Wave  | Significant tasks (t-test)  | Effect sizes        |
|-------|-----------------------------|---------------------|
| Delta | BL01*, BL02*, BL03**, BL04* | Large (d ≈ 1.3–1.7) |
| Theta | BL02*, BL03*                | Large (d ≈ 1.2)     |
| Alpha | None                        | Small–medium        |
| Beta  | None                        | Small               |

`*` p < .05 · `**` p < .01 — Music group showed lower Delta and Theta frontal power during tasks.
Baseline (IBOE) was non-significant across all bands, confirming initial group equivalence.
